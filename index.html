Summary: Latitude vs. X

This study examines the changes in weather patterns as you move closer to the equator. To conduct this study, data for over 500 cities was extracted from a database using the OpenWeatherMap API.

Once the data was gathered, it was used to create a dataframe. From there, we used data visualization libraries to examine the impact of four weather factors with respect to latitude. These four factors were temperature, cloudiness, wind speed, and humidity. This site provides the end user with descriptions, patterns, trends, and analytics of concluded from the source data upon selecting a particular factor.
